# Classify 

The is an online marketplace where people and buy, sell ,used and new items.
 
 

## Function

- User Dashboard/ Profile
- Authentication and Authorization
- User Inbox

 ## Technologies

- Python Django
- Bootstrap 
- Alpine JS
- htmx 
- DRF
- Docker 
- Postgresql
